# Operations Management System & Client Portal

This repository contains the core infrastructure for a custom ERP and Client Portal designed to automate the operational lifecycle of an international tourism venture.

## 🚀 The Business Problem
The enterprise faced significant scaling challenges due to manual scheduling and fragmented data management for a 100% international clientele. This system was engineered to move the business from spreadsheets to a centralized, logic-driven digital operation.

## 🛠️ Key Features
* **Bilingual Onboarding Portal:** A secure, mobile-friendly interface for international client registration and document management.
* **Smart Scheduling Engine:** A custom logic layer that manages precise 30-minute booking intervals, preventing overlaps and optimizing resource allocation.
* **Database Infrastructure (SQL):** Extensive use of SQL for data migration, automated financial reporting, and persistent storage of client records.
* **Financial Module:** Centralized tracking of payments, deposits, and outstanding balances.
